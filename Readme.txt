# MastGif

MastGif programı, sadece Windows 7 üzerinde çalışarak masaüstü duvar kağıdınızda hareketli resimler (Gif) oynatabilmenizi sağlar. Kullanımı basittir. İçerisinde 40'tan fazla hareketli resim bulunmaktadır; dilerseniz kendi indirmiş olduğunuz hareketli resimleri de kullanabilirsiniz.

Program başladıktan sonra sağ alt kısımda bulunan ikona sağ tıklayarak seçeneklere ve ayarlara ulaşabilirsiniz. Eğer programı açarken yetkilendirme hatası alırsanız masaüstüne kurmayı deneyin.

Orijinal olarak 2015 yılında oluşturulmuştur.
